## Screen Recording Instructions

### Setup
1. **Clean desktop** - hide all icons, close unnecessary apps
2. **Browser** - Use Chrome/Edge in incognito, bookmarks bar hidden
3. **Zoom** - Set to 125% for better readability
4. **Cursor** - Use a large, visible cursor (macOS: Accessibility settings)

### Recording Tools
- **macOS**: QuickTime Player (File ‚Üí New Screen Recording)
- **Windows**: OBS Studio or Xbox Game Bar
- **Linux**: OBS Studio

### Recording Tips
1. **Pause between actions** - 1-2 seconds for clarity
2. **Type slowly** - viewers need to read
3. **Use transitions** - simple fades between scenes
4. **Test audio** - clear voiceover, no background noise

### Post-Production
- **Trim pauses** - remove dead air
- **Add captions** - key commands/text
- **Background music** - subtle, upbeat (optional)
- **Export**: 1080p, 30fps, MP4 format

---

## Voiceover Recording Tips

### Equipment
- Use a dedicated microphone (not laptop mic)
- Record in a quiet room
- Use a pop filter if available

### Delivery
- Speak clearly and confidently
- Match pacing to visuals
- Leave natural pauses
- Record multiple takes

### Free Voiceover Tools
- **macOS**: GarageBand
- **Windows**: Audacity
- **Online**: Clipchamp, VEED.io
